HTML and CSS

Basic HTML inital day steps

Initial intro

  • Front end is the code that runs client side
  • Back end is the code that runs server side
  • Introduce the sections used to create a website:
    • HTML - creating content
    • CSS - styling the HTML
    • Javascript - making the page interactive
  • Latest versions of these:
    • HTML 5
    • CSS 3

Start coding

  • Open Visual Studio
  • Download Live Server - do within VSCode
  • Load terminal in VSCode create a directory and cd in
  • Use touch to make a file called index.html
  • Right click index.html and Open in Live Server
  • Make sure to leave this tab open as everytime we save it will update
  • Go over basics or angle brackets
  • Within index.html and type ! - this should load this HTML
